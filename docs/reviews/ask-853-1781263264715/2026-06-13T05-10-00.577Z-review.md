## Executive Summary

This branch adds append-only prompt history persistence across caption generation, alt-text, and photo-edit pipelines, enables SQLDelight migration verification via committed schema snapshots, and introduces a Jules automated code review workflow. The persistence wiring and migration artifacts are internally consistent and well-tested. However, two substantive issues weaken the integrity guarantees the branch aims to provide: on alt-text failure, the persisted caption history entry holds a resultReference pointing at a caption_results row that is never inserted (a guaranteed dangling pointer in the normal failure path, not an edge case), and the photoEditPromptHistory view silently drops entries from multi-edit chains beyond the most recent step, directly undermining the 'view photo edit history' feature. Combined with two unverifiable CI externalities and no product decision recorded for the successful-caption-discarded behavior, the branch needs targeted fixes before merging.
